# Anti Aircraft Missile (防空導彈系統)

一個基於 Minecraft 原版生存與指令設計的自動化防空攔截系統。本資料包實作了高效率的 **4 單元 4 聯裝防空導彈系統**，能有效應對中高空威脅。

---

## 核心特色

* **強大火力配置：** 完整實作「4 單元 4 聯裝」架構，單一系統最大可同時鎖定並攔截 **16 個高空目標**。
* **夜魅剋星：** 自動搜索 200~1000 格內的夜魅，並發射導彈給予攔截，維護基地領空安全。
* **反玩家高空高速突防：** 具備進階鎖定機制，可透過資料包設定或指令指定，對**使用鞘翅高空高速突防的玩家**進行有效攔截與打擊。

---

## 成就系統

* **俄羅斯空軍** ── *「今天天氣不錯，等等，那是什麼聲音？」*
  * **解鎖條件：** 被防空雷達鎖定。
* **美利堅亡牌飛行員** ── *「我以為我能閃得過...」*
  * **解鎖條件：** 被防空導彈炸死。

---

## 系統需求

* **遊戲版本：** Minecraft Java Edition `1.21.1`
* **分類：** 資料包 (Data Pack)
* **環境相容性：** 相容 **原版** 與 **Fabric** 環境。
  * *註：在 Forge、NeoForge 環境下執行時，可能會遇到箭矢渲染失效的視覺問題，但不影響實際導彈傷害與追蹤判定。*

---

## 運作機制

### 1. 多目標同時攔截
當雷達站搜索範圍內出現大量目標時，導彈會自動分流鎖定，達成最大 16 隻目標的飽和打擊能力。

### 2. 鞘翅反制防禦
針對「使用鞘翅高空高速突防的玩家」，本系統實作了提前量演算法，優化了彈道預測與追蹤半徑，但仍有攔截失敗的可能性。
* *攔截率 70 %的話，發射 3 發導彈就 210 %了嘛~*

---

## 安裝說明

1. 下載本專案的原始碼。
2. 將資料包放入你的 Minecraft 地圖路徑中：
   `saves/你的世界/datapacks/anti_aircraft_missile (本資料包)`
3. 進入遊戲中輸入 `/reload` 重新載入，或重啟伺服器即可啟用。
